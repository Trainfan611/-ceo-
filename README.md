# LEGO Store landing

Минимальный SEO-лендинг под запросы про магазин LEGO.

## Локальный запуск

```bash
npx serve -s . -l 3000
```

## Деплой в Railway

1. Создайте новый проект на Railway и подключите эту папку.
2. Назовите сервис с включением `lego` (например `lego-store`), чтобы получить URL вида:
   - `https://lego-store.up.railway.app`
3. Railway автоматически подхватит `npm start`.
4. После деплоя замените домен в:
   - `index.html` (`canonical`, `og:url`, JSON-LD `url`)
   - `robots.txt` (`Sitemap`)
   - `sitemap.xml` (`loc`)
5. Добавьте сайт в Яндекс Вебмастер и отправьте sitemap:
   - `https://ВАШ-ДОМЕН/sitemap.xml`

## Важно про "первое место" в Яндексе

Гарантировать первое место нельзя, потому что ранжирование зависит от конкурентов, поведенческих факторов и ссылок.  
Этот шаблон делает техническую SEO-базу, чтобы сайт быстрее индексировался и имел лучший шанс на высокий результат.
